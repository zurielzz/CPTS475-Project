# Analysis of the Social Impact of LLMs at WSU

**CPTS 475: Data Science (Fall 2025)**
**Team:** Zuriel Hernandez & Nicholas Vendeland

---

## Project Description

This project analyzes the social impact of Large Language Models (LLMs) on Washington State University students. We collected survey responses and performed a 5-level analysis to identify student sentiments, key predictive factors, and distinct user groups.
